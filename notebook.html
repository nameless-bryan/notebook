CSS: Display Properties

Display Properties
block: CANNOT share a line with the blocks next to it CAN have width controls
inline: CAN share the line with inline elements next to it CANNOT control the width (auto width)
inline block: CAN share the line with inline blocks next to it CAN control the width
none: Does not display, the element is removed from the flow of the document
hidden: (*) Does not display, but the element still has spaced removed in document (not removed from flow) (*)Technically, this is a visual style and not a "proper" display property

Default Presentation Styles

  1. Content - content determines the default height of the element.
  2. Order - default order comes from the code.
  3. Children sit on parents - By Default, all elements that are children will render on top of Z axis
